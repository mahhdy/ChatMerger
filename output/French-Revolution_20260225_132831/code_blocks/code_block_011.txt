# بخش ششم: میراث انقلاب فرانسه

## ۶.۱ میراث ایدئولوژیک

### مفاهیم ماندگار

| مفهوم | معنا | تأثیر |
| --- | --- | --- |
| **حقوق بشر** | حقوق ذاتی و جهانی | بنیاد حقوق بین‌الملل |
| **حاکمیت ملی** | ملت منشأ قدرت | دموکراسی مدرن |
| **شهروندی** | عضویت برابر در ملت | پایان رعیت |
| **سکولاریسم** | جدایی دین و دولت | لائیسیته |
| **ناسیونالیسم** | ملت به عنوان هویت | دو لبه (آزادی‌بخش و خطرناک) |

## ۶.۳ میراث متناقض

### تناقض بنیادین انقلاب

| آزادی | ↔ | ترور |
| --- | --- | --- |
| حقوق بشر | ↔ | استعمار |
| برابری | ↔ | محرومیت زنان |
| دموکراسی | ↔ | دیکتاتوری |

### دو روایت از انقلاب

| روایت لیبرال | روایت رادیکال |
| --- | --- |
| ۱۷۸۹ خوب، ۱۷۹۳ بد | ۱۷۸۹ ناقص، ۱۷۹۳ تکمیل |
| انقلاب باید متوقف می‌شد | انقلاب باید ادامه می‌یافت |
| ترور انحراف بود | ترور ضرورت بود |
| توکویل، فوره | ماتیه، سوبول |
