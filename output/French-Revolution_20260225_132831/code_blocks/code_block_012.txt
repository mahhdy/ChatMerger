## درس‌های کلیدی

| درس | توضیح |
| --- | --- |
| انقلاب پرهزینه است | ترور، جنگ، بی‌ثباتی |
| نهادسازی زمان می‌برد | ۱۷۰ سال تا ثبات |
| ایده‌ها مهم‌اند | حقوق بشر ماندگار شد |
| تعادل قوا ضروری است | رئیس جمهور قوی + مجلس |
| فرهنگ سیاسی کلیدی است | پذیرش دموکراسی تدریجی بود |
