🎯 محتوای این فصل:

۱.۱ **مقدمه: پیش از چپ و راست**
    • نظام‌های سیاسی پیشامدرن
    • غیاب مفهوم طیف سیاسی
    
۱.۲ **انقلاب فرانسه: لحظه زایش**
    • مجلس ملی ۱۷۸۹
    • چرا راست نشستند و چرا چپ؟
    • ژیروندن‌ها vs ژاکوبن‌ها
    
    📊 [تایم‌لاین: رویدادهای کلیدی ۱۷۸۹-۱۷۹۹]
    📊 [نمودار: چینش صندلی‌ها در مجلس]

۱.۳ **شخصیت‌های کلیدی**
    • ادموند برک (راست محافظه‌کار)
    • توماس پین (چپ رادیکال)
    • روبسپیر
    • میرابو
    
    🖼️ [کادر شخصیت: مقایسه برک و پین]

۱.۴ **گسترش مفهوم در قرن نوزدهم**
    • انقلاب‌های ۱۸۴۸
    • ظهور سوسیالیسم
    • لیبرالیسم کلاسیک
    
۱.۵ **اختلافات اولیه درون هر جناح**
    • راست: سلطنت‌طلبان vs مشروطه‌خواهان
    • چپ: جمهوری‌خواهان vs سوسیالیست‌ها
    
    📊 [جدول: مقایسه گرایش‌های اولیه]

    • پیش‌درآمد مدل‌های چندبعدی
