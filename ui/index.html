📖 Help & Guide

1
Export chats
ChatGPT → Settings → Export. Claude → Settings → Export. You'll get .json or .zip.
2
Upload
Click ⬆ Upload in Process tab, or drag .zip into the ZIP zone.
3
Select & configure
Click files to select (blue). Set collection name. Choose Batch or Merge mode.
4
Run pipeline
Click ▶ Run Pipeline. Watch live log below. Each step shows its output.
5
Get results
Click 📊 Results on the job card. Or browse Outputs tab.

🔧 Processing Modes

📋
Batch Pipeline
Each file independently: convert_export.py → chat_merger.py → post_process.py. One .md per input.
🔗
Merge Into One
All files concatenated first, then merged as one long conversation. Use for multi-part exports of the same chat.

⚙️ Options Reference

Option Default Description
Collection Name batch_… Folder name inside output/. Timestamp appended automatically.
.md / .mdx .md Output format: standard Markdown or MDX (for Next.js).
Flag Overlaps ON Mark detected repeated content with ⚠️ blocks.
Verbose OFF Pass --verbose for detailed per-segment logs.
Skip Convert OFF Skip convert_export.py if input is already standard format.
Skip Post-Process OFF Skip cleanup/TOC/validation after merge.
Extra Args — Custom CLI arguments for chat_merger.py.
