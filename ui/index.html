📖 Help & Guide

1
Export your chats
From ChatGPT: Settings → Data controls → Export data. From Claude: Settings → Account → Export data. You'll get a .json or .zip file.
2
Upload to Source folder
In the Process tab, click ⬆ Upload or drop a .zip archive into the ZIP zone. JSON files appear in the source list.
3
Select files & set options
Click files to select them (highlighted blue). Set the Collection Name, choose Batch Pipeline or Merge Into One, and toggle advanced options.
4
Run & watch the log
Click ▶ Run Pipeline. A live job card appears below showing each step's output in real-time. Click 📋 Log at any time to see full detail.
5
Find your output
When complete, click 📊 Results on the job card to see stats, or visit Outputs tab. Each collection has its .md files ready to use.

🔧 Processing Modes Explained

📋
Batch Pipeline
Each selected file goes through the full 3-step pipeline independently:
  1. convert_export.py — normalises ChatGPT/Claude JSON into a unified format
  2. chat_merger.py — identifies continuation messages, removes overlapping segments, merges assistant turns
  3. post_process.py — cleans blank lines, fixes code blocks, generates Table of Contents, validates structure
Output: one .md file per input JSON.
🔗
Merge Into One
All selected files are concatenated into a single message stream before merging. Use this when you have exported the same conversation in multiple chunks (e.g., khomeini_ch1.json + khomeini_ch2.json). Output: one combined .md file.

⚙️ Options Reference

Option Default Description
Collection Name batch_… The folder created inside output/. A timestamp is appended automatically to avoid collisions.
Output Format .md Whether to write standard Markdown (.md) or MDX (.mdx for Next.js / Docusaurus).
Flag Overlaps ON When the merger detects repeated content between segments, it inserts a ⚠️ WARNING block for manual review rather than silently discarding.
Verbose Logging OFF Passes --verbose to chat_merger.py for per-segment debug output in the job log.
Skip Convert Step OFF Skips convert_export.py. Use if your JSON is already in the standard {"messages":[...]} format.
Skip Post-Process OFF Skips post_process.py. Raw merged output only — useful for debugging or if you handle formatting yourself.
Extra CLI Args — Any additional arguments appended directly to chat_merger.py, e.g. --min-overlap 0.4 --max-gap 5.
